# WIX1002 Fundamentals of Programming (University of Malaya)

This repository contains materials related to the course **WIX1002 Fundamentals of Programming**, offered at the University of Malaya.

This course covers problem solving and the fundamental of programming. These include problem solving techniques, the basic structure of computer program, the fundamental concepts of object-oriented programming, file read write and exception handling, function, array, string, selection control structurs, repetition control structures, and data types and operations.
